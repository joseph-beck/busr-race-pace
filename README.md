# *`BUSR RACE PACE`*

## *`Dependencies`*

- numpy
- curses
- matplotlib
- datetime
- csv

## *`Usage`*

1. If you have not installed the dependencies use: \
$ make install

2. Run the app \
$ make run

## *`Example CSV`*

```
DriverName, LapNumber,  Lap,        S1,     S2,     S3,     Tyres
driver,     1,          1:49.386,   37.159, 37.238, 34.988, Hard
driver,     1,          1:26.728,   29.112, 28.637, 28.978, Hard
```
